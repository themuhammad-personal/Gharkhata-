# ঘরখাতা v1.0.0 - First Release

## ✨ Features
- Household expense tracking in Bengali
- Category-wise transaction management
- Monthly reports and analytics
- Todo list for reminders
- Local storage (no internet needed)
- Android app with WebView wrapper
- Progressive web app on Vercel

## 🚀 Installation
1. Download APK from Releases
2. Install on Android device
3. Grant installation permissions
4. Start tracking expenses!

## 📱 Platforms
- Web: https://accounting-application-design.vercel.app
- Android: Ghorkhata-v1.0.0.apk (from Releases)
